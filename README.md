# Arduino Projects

This repository serves as a record of my learning journey in electronics and microcontroller programming.

---

## Project 1: Sequential LED Blink

### Objective
To understand the fundamentals of circuit building and basic Arduino programming by controlling three LEDs in a specific sequence.

### Skills Applied
* **Electronics:** Understanding semiconductor properties (LEDs) and basic circuit completion.
* **Programming:** Learning the structure of `setup()` and `loop()` functions.
* **Control:** Utilizing `digitalWrite()` to manage power delivery to output pins.

### Hardware Components
* Arduino Board
* 3x LEDs
* 3x Resistors
* Jumper wires and Breadboard

### Code Summary
This program defines pins 11, 12, and 13 as `OUTPUT`. It executes a sequential blinking pattern using `digitalWrite()` and `delay()` commands to control the timing of each flash.

## Project 2: 4-Bit Binary Counter

### Objective
Implementing a 0 to 15 binary counter.

### Skills Applied
* **Electronics:** Managing parallel GPIO output states.
* **Programming:** Mastering digital logic and fixed-interval timing.
* **Control:** Utilizing `digitalWrite()` to represent binary states (0000 to 1111).

### Hardware Components
* Arduino Board
* 4x LEDs
* 4x Resistors
* Jumper wires and Breadboard

### Code Summary
This program defines pins 2, 3, 4, and 5 as `OUTPUT`. It iterates through binary states 0 through 15 with a 1-second delay between each state, visually representing binary counting.
